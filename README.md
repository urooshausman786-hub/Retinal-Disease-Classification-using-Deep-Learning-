# Retinal-Disease-Classification-using-Deep-Learning
Deep learning-based retinal disease classification using CNN.


## 📌 Overview
This project presents a deep learning-based framework for automated classification of retinal diseases using fundus images. After reviewing over 100 research and review papers, including clinical-based, AI-based, and optimization-based approaches, we concluded that optimization techniques significantly improve model performance and generalization.

A DenseNet201-based Convolutional Neural Network (CNN) is implemented and enhanced using optimization strategies, along with Grad-CAM for model interpretability.

---

## 🎯 Objectives
- Develop an accurate retinal disease classification model
- Compare clinical, deep learning, and optimization-based approaches
- Improve performance using optimization techniques
- Provide explainability using Grad-CAM visualization

---

## 🧠 Methodology

### 1. Literature Review
- Clinical approaches
- Deep learning models
- Optimization-based techniques (selected approach)

### 2. Model Architecture
- Pretrained DenseNet201
- Fine-tuned for retinal dataset

### 3. Optimization Techniques
- Learning rate tuning
- Regularization
- Data augmentation
- Model ensemble insights

### 4. Explainability
- Grad-CAM heatmaps for visual interpretation

---

## 📊 Results
- High classification accuracy on test dataset
- Improved performance with optimization techniques
- Clear visual explanations using Grad-CAM
